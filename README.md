# 끌올(Krawl)

> 세상에는 새로운 콘텐츠가 (말 그대로) 넘쳐납니다.<br>
> 우리는 그로 인해 외면받는 부분을 조명하고자 합니다. 발행된지는 오래되었지만, 그냥 묻히기에는 아까운 콘텐츠들 말입니다. 마치 고전 작품처럼요.
